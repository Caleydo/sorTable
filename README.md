=====================

table visualization that allows sorting, aggregation, filtering, hiding and re-ordering

Installation
------------

```
git clone https://github.com/cnobre/sorTable.git
cd sorTable
npm install
```

Testing
-------

```
npm test
```

Building
--------

```
npm run build
```
